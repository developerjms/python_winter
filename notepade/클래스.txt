클래스 : 형틀
객체 : 클래스로 부터 생성된 실체

붕어빵틀(클래스), 붕어빵(객체), 반죽,앙꼬(매개변수)

함수 : 반복되거나 특정 기능을 분리하기 위해 사용
객체 : 데이터+함수(==>메서드)

예)
DEL 개체(변수, 값...) : 명령
DEL (매개변수) : 함수
A.DEL(매개변수) : 객체A의 메서드

메세지 : 객체에서 객체로 전달하는 데이터
외부에서 들어오는 자극
인스턴스 : 각 클래스로 부터 생성된 객체A
상속 : 상위 클래스의 내용을 하위 클래스에 물려받는 것.

오버로딩 : 함수명 같음, 기능이 유사하지만 다를때 사용
오버라이딩 : 상속될때 사용, 부모의 메서드가 있음, 유사하지만 약간 다르다. 부모에게있는 것을 자식에 맞추어 사용
생성자 : 객체가 생성될 대 자동으로 실행되는 메서드로 해당 객체가 가져야할 기본적인 설정을 위해 사용.

객체 : 본체
인스턴스 : 분체
